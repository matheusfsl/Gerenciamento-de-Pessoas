#  API de Gerenciamento de Pessoas

Este projeto é uma **API RESTful desenvolvida com Spring Boot** para o gerenciamento de informações de pessoas.  
Permite **criação, atualização, exclusão lógica, busca individual ou em lote**, e também **filtragem personalizada** por atributos como nome, altura, peso e estado.

---

##  Tecnologias Utilizadas

- **Java 17+**
- **Spring Boot 3+**
- **Spring Web**
- **Spring Data JPA**
- **Hibernate**
- **Jakarta Validation**
- **MySQL (ou outro banco compatível)**
- **Lombok**
